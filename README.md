## 📈 Troubleshooting Common Issues

Sometimes, the application might stop responding. Here are steps to get back on track:

*   Scan Stuck at Zero Percent: Check your firewall settings. Sometimes, the Windows firewall blocks the internal connection the app uses to scan.
*   Missing Data: If you cannot find your reports, look inside the Spectremind folder for a sub-folder labeled History. Your files reside there.
*   Command Not Recognized: Ensure you type only the number requested by the prompt. Extra letters or symbols cause errors.

If you encounter persistent issues, close the window completely and restart the application. The software resets its connection every time you open it.
